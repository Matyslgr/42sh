# 42sh
End of 1st year project at Epitech Montpellier.
Creation of a Shell like the TCSH.

* Execution of programs
* Pipes, Redirects
* Historical
* Backticks
* Variables (Environment, Local)
* Line editing
* A.k.a
* Inhibitors
* Globbings
* Order of Priority of Command Line Arguments

## How to Launch

```bash
make re && ./42sh && make fclean
```
